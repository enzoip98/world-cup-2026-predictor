# Mundial 2026 ⚽

Aplicación web para gestionar asistencia a partidos, registrar pronósticos y calcular puntajes durante el Mundial FIFA 2026.

## Características

- 📅 Calendario completo de partidos
- 🎟️ Registro de asistencia
- 📊 Pronósticos de resultados
- 🏆 Sistema de puntuación automático
- 👥 Visualización de asistentes por partido
- 🔒 Autenticación con Firebase
- ☁️ Persistencia en Firestore
- 📱 Diseño responsive

## Tecnologías

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore

## Instalación

```bash
git clone https://github.com/tuusuario/mundial-2026.git

cd mundial-2026

npm install

npm run dev
```
